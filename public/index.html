How it Works

  1. Receiver creates a session
  2. QR code is generated and displayed
  3. Sender scans QR code with their device
  4. Files are uploaded securely to cloud storage
  5. Receiver gets real-time progress updates
  6. Files can be downloaded with secure links

✨ Features

🔒 Secure: Files stored in cloud with signed URLs
⚡ Real-time: Live progress updates via Server-Sent Events
📤 Large Files: Chunked uploads support files up to 100MB
🧹 Auto-cleanup: Sessions automatically expire and cleanup
